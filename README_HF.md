# Mitologi Clothing - AI Recommendation Service

This service provides machine learning-powered product recommendations for the Mitologi Clothing platform.

## 🚀 Deployment Info
- **SDK:** Docker
- **Port:** 7860
- **Base Image:** python:3.11-slim

## 🔐 Security
Don't forget to set the `RECOMMENDER_API_KEY` in the Hugging Face Space **Settings > Variables and Secrets** section.

## 🛠 Tech Stack
- Flask & Waitress
- Scikit-Learn
- Pandas
